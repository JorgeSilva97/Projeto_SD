Please follow the following steps to test the _01_helloworld example:

1. Create a Java project called *SD* (e.g. inside the Projects folder)
2. Unzip the *edu.ufp.inf.sd._01_helloworld* package/folder into the *src* folder
3. Use IntelliJ/NetBeans to compile the package *edu.ufp.inf.sd._01_helloworld*
4. Update de setenv (.sh/.bat) file according to your environment (Mac/windows)
- runpython
- runregistry
- runserver
- runclient
5. Run the python http server (script *_1_runpython*)
6. Run the rmi name service (script *_2_runregistry*)
7. Run the server/servant (script *_3_runserver*)
8. Run the client (script *_4_runclient*)
